Background Images

Property 1: background-image

E.g. {background-image: url('...'); }

Property 2: background-color

E.g. { background-color: green; }

Property 3: background-repeat

repeat, repeat-x, repeat-y and no-repeat.

Property 4: background-position

Top, Right, Left, Bottom and Center.
